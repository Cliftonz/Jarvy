# Jarvy Introduction

Welcome to Jarvy, the ultimate tool for setting up your local development environment effortlessly!
Jarvy takes the opposite approach of many configuration tools and allows you to install dependencies right on the computer.
Many companies do not have the resources to configure dev instances, install dependencies on cloud images. 
So we take the idea and automate it given that your laptop is already the dev environment.
Jarvy automates the installation and configuration of essential development tools by using a simple and intuitive configuration file.

## Why
Jarvy was developed out of the need to help users configure their computers to contribute to open source projects.
We found that many developers that wanted to contribute, could not get there computer set up for the project to work.
While there are options like Github [Codespaces](https://github.com/features/codespaces) they are not usable without a quality internet connection.
This is why we want to make sure that every project can easily set up contributors computers to help them contribute. 

## Key Features

- **Simple Configuration**: Define your environment setup with a straightforward TOML configuration file.
- **Flexible Tool Installation**: Specify tool versions and whether to use a version manager for installation like nvm for node.
- **Cross-Platform Support**: Works seamlessly on Windows, macOS, and Linux, leveraging platform-specific package managers.
- **Easy to Use**: Run `jarvy setup` to automatically configure your computer.

### Why Use Jarvy?

Setting up a development environment can be time-consuming and error-prone, especially when collaborating with a team or working on multiple projects. Jarvy eliminates these challenges by providing:

- **Consistency**: Ensure all team members have the same tools and versions installed.
- **Simplicity**: Reduce setup time with a single configuration file.
- **Flexibility**: Customize installations based on project requirements.
- **Efficiency**: Automate repetitive tasks and focus on development.
